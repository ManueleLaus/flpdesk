Come configurare i ruoli
I ruoli vengono gestiti direttamente nel file app.html nella sezione const ROLES = {…}.

Per aggiungere un Super Admin o un Editor, aggiungi la loro email nelle rispettive liste. Tutti gli altri utenti saranno automaticamente Lettori.

Dopo la modifica, carica il file aggiornato su GitHub e Netlify si aggiornerà automaticamente.
